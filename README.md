## for start project

```js
    // install dependencies
    npm install

    // for locally run json db
    npm run dev:db

    // for locally run next app
    npm run dev:web
```

## all commands

```js
    // locally run
    npm run dev
    npm run dev:db
    npm run dev:web
    npm run dev:react-web
    npm run dev:shared-ui

    // build
    npm run build
    npm run build:web
    npm run build:react-web
    npm run build:shared-ui

    // for start
    npm run start
    npm run start:db
    npm run start:web
    npm run start:react-web
    npm run start:shared-ui
```
